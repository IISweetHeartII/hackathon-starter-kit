# AI Template

Use this folder as a guide for AI feature integration.

## Quick Start Checklist

- Define one clear user problem AI should solve
- Select model/provider and fallback behavior
- Design prompt strategy and response format
- Add safety filters and output validation
- Prepare non-AI fallback UX for demo stability

## Suggested Baseline

- Prompt versioning file (`prompts.md`)
- Guardrails and refusal handling
- Latency timeout and retry policy
- Example test prompts for regression checks
