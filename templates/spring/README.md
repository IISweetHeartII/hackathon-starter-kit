# Spring Boot Template

Use this folder as a guide for backend setup with Spring Boot.

## Quick Start Checklist

- Define bounded contexts and core entities
- Create MVP API endpoints and validation
- Configure profile-based env handling
- Add health check and structured logging
- Prepare demo-safe fixture data

## Suggested Baseline

- Consistent error response schema
- Request DTO validation for write endpoints
- Basic integration test coverage for critical flow
- README with local run + test commands
