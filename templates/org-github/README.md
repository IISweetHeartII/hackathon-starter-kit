# Org .github Template

This template helps bootstrap an organization-level `.github` repository.

## Included Files

- `profile/README.md` - Org profile landing page
- `ISSUE_TEMPLATE/bug_report.md` - Bug report template
- `ISSUE_TEMPLATE/feature_request.md` - Feature request template
- `pull_request_template.md` - Pull request checklist
- `CODEOWNERS` - Review ownership baseline

## How to Use

1. Create a repository named `.github` under your org.
2. Copy this template into that repository.
3. Customize project links, owners, and workflow rules.
