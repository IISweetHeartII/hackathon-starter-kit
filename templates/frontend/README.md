# Frontend Template

Use this folder as a guide when creating the actual frontend app.

## Quick Start Checklist

- Decide framework: React / Next.js / Vue
- Set routing and base layout
- Configure API client and env handling
- Add auth flow if needed
- Prepare demo-safe seed data

## Suggested Baseline

- TypeScript
- Lint + format settings
- Component folder conventions
- Error/loading UI states for all main pages
