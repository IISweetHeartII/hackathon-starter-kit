# Next.js Template

Use this folder as a guide for frontend setup with Next.js.

## Quick Start Checklist

- Initialize Next.js app with TypeScript
- Set app router and shared layout
- Add API client and environment strategy
- Add auth gate and role-aware route handling if needed
- Prepare demo-safe mock/fallback data

## Suggested Baseline

- TypeScript strict mode
- Lint + format + pre-commit checks
- Feature-first folder conventions
- Error/loading/skeleton states for all key pages
