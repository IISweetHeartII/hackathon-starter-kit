# NestJS Template

Use this folder as a guide for backend setup with NestJS.

## Quick Start Checklist

- Define module boundaries early
- Set DTO validation with class-validator
- Configure env and secret handling
- Add health endpoint and request logging
- Prepare deterministic demo fixtures

## Suggested Baseline

- Exception filters for consistent error responses
- Pipe-based input validation for all write APIs
- Basic e2e test for main user journey
- README with local run + test commands
