# Backend Template

Use this folder as a guide for backend service initialization.

## Quick Start Checklist

- Choose stack: Spring Boot / FastAPI / NestJS
- Define domain model and core API endpoints
- Set environment config and secrets policy
- Add health endpoint and basic logging
- Add demo-safe fallback data

## Suggested Baseline

- Consistent error response schema
- Request validation for all write APIs
- Basic observability (request logs, error logs)
- README with local run instructions
