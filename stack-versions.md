# Stack Versions (Recommended Baseline)

These are practical defaults for hackathon speed and stability.

## Frontend

- Node.js: 20.x LTS
- Next.js: 15.x
- TypeScript: 5.8+
- Tailwind CSS: 4.x

## Backend

### Spring Boot Track
- Java: 21 LTS
- Spring Boot: 3.5.x
- Gradle: 8.10+

### NestJS Track
- Node.js: 20.x LTS
- NestJS: 11.x
- TypeScript: 5.8+

### FastAPI Track
- Python: 3.11+
- FastAPI: 0.116.x
- Uvicorn: 0.35.x

## AI Integration

- Vercel AI SDK (optional in JS stack)
- OpenAI / Anthropic / Gemini SDKs
- Always include demo-safe fallback path

## Infra

- Postgres 16+
- Redis 7+

## Versioning Rule

During active hackathon work, prioritize stable major versions and avoid major upgrades.
